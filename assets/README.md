# Assets structure

- `assets/css/style.css` — simplified global design system.
- `assets/js/includes.js` — loads `includes/header.html` and `includes/footer.html` into pages.
- `assets/js/main.js` — menu toggle, active nav link and footer year.
- `assets/images/brand/` — logo and brand graphics.
- `assets/images/places/` — place and ghat images.
- `assets/images/rituals/` — ritual/service images.
- `assets/images/kathas/` — katha-related images.
- `assets/images/people/` — acharya/guide/profile images.
- `assets/images/placeholders/` — safe placeholder graphics.
- `assets/videos/` — video files, not mixed with images.
